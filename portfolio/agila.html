~sgmaglanque @ Agila

In college, we were given our own personal webspace in the university server as Computer Science students. It's mostly used to deploy various projects from different subjects where we can showcase to our professors what we have learned and that we were able to apply them.

The website have had multiple versions through my college years, drawing inspiration from my blockmates and other web developers' sites I've explored online. When I started working on my thesis, I've decided to just turn it into a simple webpage, with all my previous projects archived on the server.

Sometimes the agila (our school server's name) server becomes down, so I made a copy of it here.
