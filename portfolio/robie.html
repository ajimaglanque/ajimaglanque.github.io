ROBIE

ROBIE is a cross-platform online banking mobile application developed to be offered to SGV clients. It has different features such as user dashboard with analytic components, transactions monitoring, bills payment, loan applications, credit score computations, and other bank-related services.

Mainly developed using React Native with the open-source platform Expo, which is designed to make universal native apps for Android, iOS, and the web. It also utilized some other Node packages heavily based on React and JavaScript.
